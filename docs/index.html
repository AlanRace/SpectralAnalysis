SpectralAnalysis

SpectralAnalysis is analysis software for spectral imaging data, with a particular focus on mass spectrometry imaging data. SpectralAnalysis provides functionality for data visualisation, preprocessing, univariate statistics, multivariate analysis and machine learning.
